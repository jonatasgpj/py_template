# Projeto FastAPI: meu_projeto

## Estrutura do Projeto

```
meu_projeto/
├── app/
│   ├── __init__.py
│   ├── main.py
│   └── routes.py
├── run.py
├── requirements.txt
└── README.md
```

## Como executar

1. Instale as dependências:
   ```bash
   pip3 install -r requirements.txt
   ```

2. Rode o servidor:
   ```bash
   python run.py
   ```

3. Acesse no navegador:
   ```
   http://127.0.0.1:8000
   ```

## Requisitos

- Python 3.8+
- FastAPI
- Uvicorn
